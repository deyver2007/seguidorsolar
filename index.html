¿Qué es?

Este seguidor solar es un panel solar que, por medio de un circuito, funciona principalmente gracias al módulo de tiempo y un fotorresistor. El módulo de tiempo da la hora en la que se debe mover el motor, mientras que el fotorresistor ayuda a evitar el consumo innecesario de energía durante la rotación del panel solar.

Objetivo

Este panel está diseñado para maximizar la recolección de energía, permitiendo acumular energía en una batería para su uso como energía de emergencia o como fuente alternativa, conectando la energía directamente a la casa para ahorrar en costos energéticos.

¿Para qué se necesitaría?

Este proyecto está centrado en ser una fuente de energía de emergencia o alternativa, ideal para ahorrar en costos de energía y tener una fuente barata y confiable.

Materiales y cómo sirven

  • ESP32: Microcontrolador con conectividad a internet.
  • DS3231: Módulo de tiempo para identificar la hora del día.
  • Cables: Elementos fundamentales para conectar el circuito.
  • Servomotor SG5010: Permite el movimiento del panel solar.
  • Una tabla: Estructura base del seguidor solar.
  • Fotorresistor: Detecta el nivel de luz y asiste en el control energético.
  • Resistencia: Protege al fotorresistor de sobrecargas.
